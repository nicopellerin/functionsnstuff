# Building a basic HTTP server in Go

<Spacer margin="6rem 0 4rem 0" />

```go
package main

import (
  "net/http"
  "fmt"
)

func helloWorld (w http.ResponseWriter, r *http.Request) {
  fmt.Fprint(w, "helloworldnstuff")
}

func main() {
  http.HandleFunc("/", helloWorld)

  http.ListenAndServe(":8080", nil)
}
```
