# Accessing environment variables (.env)

<Spacer margin="6rem 0 4rem 0" />

```jsx
// In gatsby-node.js
require('dotenv').config()

// or when using another name than .env (eg: .env.production)
require('dotenv').config({
  path: `.env.${process.env.NODE_ENV}
})
```
