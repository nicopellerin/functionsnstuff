# Printing vector to console

<Spacer margin="6rem 0 4rem 0" />

```rust
// To print a vector to console, use the `debug` trait {:?}
let vector = vec!["Nico", "Alexe", "Val"];

println!("{:?}", vector); // ["Nico", "Alexe", "Val"]
```
