# Split an array in half

<Spacer margin="6rem 0 4rem 0" />

```jsx
// Array of 7 elements
const array = [1, 2, 3, 4, 5, 6, 7]

// Calculate the half the length of the array.
const halfLength = Math.ceil(array.length / 2)

// Using .slice doesn't mutate the original array
const firstHalf = array.slice(0, halfLength)
const secondHalf = array.slice(halfLength)

console.log(firstHalf) // [1,2,3,4]
console.log(secondHalf) // [5,6,7]
```
