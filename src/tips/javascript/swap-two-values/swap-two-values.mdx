# Swap two values in Javascript

<Spacer margin="6rem 0 4rem 0" />

```jsx
let dog = "Max"
let cat = "Nini"

let temp = cat
cat = dog
dog = temp

// Using array destructuring
[dog, cat] = [cat, dog]

// Result -> dog = "Nini", cat = "Max"
```
